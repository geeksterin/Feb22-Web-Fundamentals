Tells the behaviour of the element

inline display:
Height and Width attribute does not work when the display is inline.
it automatically takes the Height and width accordin gto the content present

Block display:
By default, block elements cover the whole width of the screen.

inline-block
it behaves like an inline element but one can add height and width to it.
